Elemen Tambahan HTML



HTML adalah singkatan dari Hypertext Markup Language.
HTML Merupakan bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web.
HTML bukanlah bahasa pemrograman, melainkan bahasa markup yang berfungsi untuk mengatur struktur dan format konten di dalam dokumen web.


1. Formulir :Tag : < from > Formulir digunakan untuk mengumpulkan data dari pengguna, seperti nama, email, atau pesan. Formulir dapat terdiri dari berbagai elemen seperti input teks (< input >), tombol kirim (< button >), dan label (< label >) untuk memberikan deskripsi pada input. Ini sangat penting untuk aplikasi web yang memerlukan interaksi pengguna, seperti registrasi atau pengisian survei.

2. Tabel :Tag : < table > Tabel digunakan untuk menyusun data dalam format grid yang terstruktur. Tag < tr > digunakan untuk baris, < td > untuk kolom, dan < th > untuk header kolom. Tabel sangat berguna untuk menampilkan data terstruktur yang membutuhkan keteraturan, seperti jadwal, harga produk, atau data statistik.
