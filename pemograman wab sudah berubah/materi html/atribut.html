Atribut dalam HTML



HTML adalah singkatan dari Hypertext Markup Language.
HTML Merupakan bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web.
HTML bukanlah bahasa pemrograman, melainkan bahasa markup yang berfungsi untuk mengatur struktur dan format konten di dalam dokumen web.


1. Atribut dalam HTML :Tag Dapat digunakan pada semua elemen sepertib : < a >, < img >, < p >, < div >, dan lainnya. Atribut memberikan informasi tambahan pada elemen HTML dan sering digunakan untuk menyesuaikan atau memberikan fungsi khusus pada elemen tersebut. Beberapa contoh atribut yang umum digunakan: href pada tag < a >: Menentukan alamat tujuan tautan. src pada tag < img >: Menentukan sumber file gambar. class atau id: Digunakan untuk memberikan identitas pada elemen, memungkinkan pengaturan gaya dengan CSS atau interaksi dengan JavaScript.
