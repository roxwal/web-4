Struktur Dasar HTML


HTML memiliki struktur hierarki yang membagi dokumen menjadi tiga bagian utama, masing-masing dengan fungsi spesifik untuk menyusun dan menyajikan halaman web. Berikut penjelasannya:.

Struktur Html

1. Deklarasi Dokumen :Bagian ini terletak di awal dokumen HTML dan digunakan untuk memberi tahu browser bahwa dokumen menggunakan versi HTML tertentu. Dalam HTML5, deklarasi ini sangat sederhana dan memastikan kompatibilitas dengan browser modern. Dalam HTML5, deklarasi ini sangat sederhana dan memastikan kompatibilitas dengan browser modern.

2. Bagian Kepala (Head) : Bagian ini mengandung informasi yang tidak langsung terlihat oleh pengguna tetapi penting bagi browser dan mesin pencari. Fungsi utama bagian ini meliputi: Judul Halaman: Menentukan nama halaman yang tampil di tab browser. Metadata: Informasi tambahan tentang halaman, seperti deskripsi, kata kunci, dan pengaturan karakter. Link Eksternal: Menyertakan file CSS untuk gaya atau skrip JavaScript untuk fungsi tambahan. Hingga saat ini versi HTML sudah mencapai versi 5 yang dirilis pada tahun 2012, Versi HTML 5 ini merupakan terusan dari perpanjangan versi HTML 4.01 yang sebelumnya digunakan oleh kebanyakan orang.

3. Bagian Tubuh (Body) : Bagian Tubuh (Body) Bagian ini berisi semua konten yang akan ditampilkan kepada pengguna di browser. Konten dalam bagian tubuh dapat berupa: Teks: Paragraf, judul, atau daftar. Gambar dan Video: Elemen multimedia untuk memperkaya halaman. Tautan dan Formulir: Untuk navigasi atau interaksi pengguna. Bagian ini adalah inti dari halaman web karena semua elemen yang terlihat berasal dari sini.
