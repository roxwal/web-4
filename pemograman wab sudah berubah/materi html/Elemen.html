Elemen Utama Dalam Html



HTML adalah singkatan dari Hypertext Markup Language.
HTML Merupakan bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web.
HTML bukanlah bahasa pemrograman, melainkan bahasa markup yang berfungsi untuk mengatur struktur dan format konten di dalam dokumen web.

Elemen Html


1. Heading (Judul) :Tag: < h1 > Hingga < h6 > Heading digunakan untuk memberikan struktur pada dokumen dengan berbagai tingkatan. Tag ini membantu menyusun hierarki informasi, dari judul utama hingga subjudul.

2. paragraf : Tag : < p > Elemen ini digunakan untuk menampilkan teks dalam format paragraf, membuatnya lebih rapi dan mudah dibaca.

3. Tautan (Link) : Tag : < a > Tautan digunakan untuk menghubungkan halaman web satu sama lain atau menuju sumber eksternal. Tautan sering memiliki atribut seperti href untuk menentukan alamat tujuan.

4. Gambar : Tag : < img > Elemen ini digunakan untuk menampilkan gambar di halaman web. Tag ini sering dilengkapi atribut seperti src untuk sumber gambar dan alt untuk deskripsi.

5. Daftar : Daftar Tidak Berurutan Tag : < ul > untuk daftar utama dan < li > untuk item dalam daftar. Digunakan untuk membuat daftar dengan simbol seperti titik.
Daftar Berurutan Tag: < ol > untuk daftar utama dan < li > untuk item dalam daftar. Digunakan untuk membuat daftar yang bernomor atau berhuruf.
