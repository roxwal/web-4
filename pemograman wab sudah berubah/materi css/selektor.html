Selector CSS


Selektor adalah cara untuk menentukan elemen HTML mana yang akan diberi gaya oleh CSS. Berikut jenis-jenis selektor yang umum:


1. Selektor Elemen: Mengatur gaya berdasarkan nama elemen HTML, seperti < p >, < div >, < h1 >.

2. Selektor ID: Mengatur elemen dengan atribut id. ID bersifat unik, sehingga hanya dapat diterapkan pada satu elemen.

3. Selektor Kelas: Mengatur elemen dengan atribut class. Kelas bisa diterapkan pada beberapa elemen.

4. Selektor Grup: Mengatur beberapa elemen sekaligus dengan memisahkannya menggunakan koma, misalnya h1, p, div.

3. Selektor Turunan: Mengatur elemen yang berada di dalam elemen lain. Contoh: div p untuk memilih < p > di dalam < div >.
